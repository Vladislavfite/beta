-папка attacktower содрежит 12 типов башн для постройки, папка attack содержатся папки tower с номерами типов башн с кадрами анимаций для каждого типа башн, в папке statik находятся также папки tower с номерами типов башн но уже с кадрами башен в спокойном состоянии
-папка elements находится moloticon.png для отображения иконок на основной карте, при нажатии на эту иконку можно будет построить башню первого типа. Также есть два элемента up для наглядной информации что башню можно улучшить до следующего типо при доступном балансе монет, и noup информирующий нас о том что улучшить тип постройки еще нельзя. Этот знак будет распологаться внизу слева возле каждой башни
- папка enemy находятся три папки с кадрами атаки врага (atack_enemy), смерти (die_enemy), и его ходьбы (walk)
- файл map.png это основная статичная карта
